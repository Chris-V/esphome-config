# Terrarium monitor
Board: ESP8266 NodeMCU v2.0

## Sensors

### [DS18B20](./datasheets/DS18B20.pdf) - Temperature sensor
_Basking temperature._

DQ: D5 / GPIO14

### [AM2315](./datasheets/AM2315.pdf) - Temperature & RH sensor
_Ambiant temperature and relative humidity._

SCL: D1
SDA: D2
